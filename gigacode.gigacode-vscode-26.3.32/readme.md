GigaCode — это AI-ассистент для разработки в IDE: генерация, объяснение и редактирование кода, включая агентный режим для выполнения задач целиком.

Возможности:

💬 `CodeChat` — чат для работы с кодом и контекстом проекта с агентным режимом;  
▶️ `Запуск команд` в терминале и сборка проектов;  
🛠️ `Рефакторинг`, `генерация тестов` и `документации`;  
🧠 Свежие AI-модели `CodeChat 235В` и `Inline v4.0 13В`  
✏️ `Inline Code Assistant` — подсказки и правки прямо в кодовой строке;  
🔤 Поддержка 35+ языков программирования  

Узнать больше о возможностях и новых релизах в [документации](https://gitverse.ru/docs/ai-assistant-gigacode/quick-start/)
